тестирование сайта https://izhflower18.ru/

сценарий 1 - сортировка по цене
Цель: провести тестирование сортировки товаров по цене

шаг 1. Открыть сайт
- открыть браузер
- Ввести URL адрес сайта
- нажать enter
шаг 2. Отсортировать цветы по цене
- дождаться прогрузки сайта
- пролистать до кнопки-ссылки "цене"
- навести курсор на кнопку
- нажать левую кнопку мыши
- дождаться прогрузки сайта
- пролистать до сетки товаров

ожидаемый результат: ожидается, что после повторного обновления страницы в сетке товаров, все букеты  будут отсортированы по цене

фактический результат: при обновлении страницы, сетка товаров была отсортирована в порядке убывания


сценарий 2 - Контакты - наличие обратной связи
Цель: проверить наличие страницы для обратной связи на сайте

шаг 1. Открыть сайт
- открыть браузер
- Ввести URL адрес сайта
- нажать enter
шаг 2. открыть страницу с контактами
- дождаться прогрузки сайта
- навестись курсором на кнопку-ссылку "контакты"
- нажать левую кнопку мыши
ожидаемый результат: при нажатии на кнопку-ссылку будет выполнен переход на страницу "контакты"

фактический результат: при нажатии на кнопку-ссылку, открылась страница "контакты"
